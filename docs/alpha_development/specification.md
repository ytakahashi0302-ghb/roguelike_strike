# Rogue Strike Alpha Specification

## 概要
Rogue Strikeのプロトタイプ（Vanilla TS + Canvas）をベースに、今後の本格的な機能拡張（UIアニメーション、複雑なステート管理、新しいキャラやスキルの追加）を見据えて、**React (Vite + React + TypeScript)** を採用したAlpha版の開発を行う。
さらに、リプレイ性を高めるためのローグライト機能「メタ・プログレッション（永続強化・スキルツリー）」を追加する。

## コアゲームループ
プロトタイプのロジックを引き継ぐ。
1. チーム編成（最大4体）
2. 敵が配置された盤面でのターン制バトル（引っ張りアクション）
3. 敵を全滅させるとクリア、報酬獲得およびコイン獲得。
4. ステージ5のボスを倒すとゲームクリア（ランの終了）。

## 新規要素：メタ・プログレッション（永続強化）
ラン中に敵を倒したりステージをクリアすることで「コイン（またはストライクポイント等）」を獲得し、ラン終了後（Start Screen）に恒久的なアップグレードを行えるようにする。

### 強化項目（スキルツリー）案
- **初期防衛ライン強化**: 敵に1回突破されてもゲームオーバーにならない「シールド」を初期装備する。
- **ベース攻撃力アップ**: 全キャラの基礎ダメージにボーナス（+1）。
- **チームキャパシティ拡張**: 最初から2体のキャラを持ってスタートできる。
- **レア報酬確率アップ**: 報酬画面で未開放・または強力なキャラが出やすくなる。

## アーキテクチャ (React移行)
- **UI層**: Start, Stage Select, Reward, Game Over, Skill Treeなどの画面はすべてReactコンポーネント（HTML/CSS）で実装する。
- **状態管理**: チーム状態、所持コイン、ステージ進行状況などのステート管理をReactフックで行う。
- **ゲームレンダリング層**: Canvas APIを引き続き使用。Reactの `useEffect` や `useRef` を用いて、ゲームのコアクラス (`GameEngine`, `Player`, `Enemy`) とReact UIの橋渡しを行う。ゲームの描画ループ（requestAnimationFrame）はReactのレンダリングサイクルから独立して動かす。
