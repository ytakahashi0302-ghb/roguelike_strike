# Rogue Strike Alpha開発フェーズ 修正内容の確認 (Walkthrough)

## 概要
Rogue Strikeのプロトタイプ版（Vanilla TypeScript + Canvas API）から、アルファ版を見据えた構成整理とメタプログレッション機能の実装を完了しました。ユーザーの要望通り、開発環境をVite + Reactに移行し、UIおよびゲームの管理をよりコンポーネント指向性の強い形に改修しています。

## 実施した変更内容

### 1. React + TypeScript環境への移行
- `index.html`から従来の直接的なUI表示の記述を落とし、Reactをマウントする`<div id="root"></div>`へと更新しました。
- `tsconfig.json`でReact JSXが動作するようコンパイラフラグを調整しました。

### 2. ゲームコアロジックのモジュール・クラス化 (`CoreEngine.ts`, `Entities.ts`)
- これまで１ファイルの`gameEngine.ts`にすべて記載されていたロジックをリファクタリングし、`Player`クラスと`Enemy`クラスを`Entities.ts`に抽出しました。
- `GameEngine`というクラスモジュールを定義し、Reactコンポーネント（UI側）とのインターフェースを用意しました。これにより、「フェーズ状態」「ターン」「コイン数」などをReactのステートと連携して更新できるようになりました。

### 3. ReactコンポーネントによるUI構築 (`App.tsx`)
- スタート画面、ステージ選択画面、報酬画面、ゲームオーバー画面を全てReactで実装しました。
- 各種画面遷移（`phase`ステートの切り替え）をフックで行い、イベントや画面表示が整理されています。

### 4. コイン獲得とスキルツリー（メタプログレッション）の実装
- ゲーム内でターン経過、または敵撃破に応じてコインを獲得するロジックを組み込みました。リザルト画面で今回のランでの獲得コイン数が表示されます。
- `localStorage`を利用して「総コイン数」と「購入済みアップグレード」を永続保存する機能を実装しました。
- スタート画面からアクセスできる「Upgrades（スキルツリー）」画面を作成し、以下の特典を実装しました：
  - **Initial Shield (50 Coins)**: 初回ダメージ無効化効果。
  - **Damage Boost (100 Coins)**: 全キャラクターの基礎ダメージ+1効果。
  - **Team Capacity (150 Coins)**: ラン開始時の初期編成枠が2体になる効果。
- 該当する能力強化が次のランをスタートした時点で`GameEngine`へと設定され、動的にメタプログレッション（永続成長）が楽しめるようになりました。

## 検証結果
- Typescriptの型チェック (`tsc`) およびViteのビルドによって、ソースコード全体のエラーが無いことを検証・確認しました。
- データの保存と読み込みは`localStorage`を通じて正常にフックされるように実装されています。
