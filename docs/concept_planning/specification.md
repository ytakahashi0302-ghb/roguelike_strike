# ゲームコンセプト仕様書: ローグライク・ストライク（仮称）

## 1. コアコンセプト
- **モンスターストライク × ブロック崩し × インベーダー**
- **操作性**: プレイヤーは自エリア（画面最下部）から自キャラを射出する。引っ張り距離によらず、**射出方向**のみを決定する。
- **勝利条件**: ステージ内の敵をすべて全滅させること。
- **敗北条件**: 敵が画面最下部の「プレイヤーエリア（防衛ライン）」に到達し、プレイヤーの全体HPが0になること。自キャラ個別のHPは存在しない。
- **進行**: プレイヤーが1回射出する（または全員射出する）ごとに、インベーダーゲームのように敵が1段（または一定距離）下へ迫ってくる。

## 2. ローグライク要素
- **ステージ進行（ラン）**: 
  - 1回のプレイ（ラン）は複数の「エリア」に分かれる。各エリアの最後にボスステージが存在。
  - 道中のステージはランダムに生成（敵の配置、種類）。
- **ステージクリア報酬**: 
  - ステージクリアごとに3つの選択肢から1つを選ぶ（Slay the Spire形式）。
  - **報酬の種類**:
    - **新規キャラの追加**: チーム（最大4体）に新しい特性を持つキャラを加える。
    - **ステータス強化**: 既存キャラの威力、スピード、サイズなどを強化。
    - **アーティファクト**: ラン全体にパッシブ効果をもたらす（例: 「最初の1撃のダメージ2倍」「防衛ラインの耐久値増加」）。
- **パーマデス**: プレイヤーHP（防衛ライン耐久値）が0になったらゲームオーバーとなり、初期状態からやり直し。

## 3. キャラクター（自キャラ）の特性
自キャラはそれぞれ異なる物理的・攻撃的特性を持つ。

### 基本ステータス
- **ダメージ（威力）**: 敵に衝突した際に与える基礎ダメージ。
- **スピード（スタミナ）**: 射出後の初速、および減衰率による移動継続能力。
- **サイズ**: 当たり判定の大きさ（当たりやすいが隙間を抜けにくい大、抜けやすい小など）。

### タイプ・特性のアイデア
1. **バウンス（反射）**: 
   - スタンダードなタイプ。敵や壁に当たると弾性で跳ね返り、狭い隙間に入り込めば大ダメージ（カンカン）を狙える。
2. **ピアス（貫通）**: 
   - 敵を貫通して直進する。敵に当たるたびに一定割合スピードが減衰する。後列の厄介な敵を直接狙うのに適している。壁では反射する。
3. **スプリット（拡散）**: 
   - 最初の敵に衝突した瞬間、進行方向の左右に2発の「子機（小ダメージ・バウンス型）」を射出する。広範囲の敵のHPを削るのに有効。
4. **ヘヴィ（重力）**: 
   - スピードは遅く移動距離は短いが、敵のノックバック（上方向への押し戻し）量が非常に大きい。防衛ラインに迫る敵を押し返すタンク役。
5. **ブラスト（爆発）**: 
   - 停止直前に大爆発を起こし、周囲の敵すべてに大ダメージを与える。位置取りが重要なフィニッシャー。

## 4. 敵キャラクターの特性
敵は基本的にターン（プレイヤーの1射出）ごとに1段（グリッド）下へ迫ってくる。

1. **歩兵**: 特殊能力はない標準的な四角いブロック。
2. **重装兵（シールド）**: 前面（下方向）からのダメージを1にする。壁から反射させて上部や側面から当てる必要がある。
3. **特攻兵（ダイバー）**: 通常1段のところ、毎ターン2段降りてくる。放置するとすぐゲームオーバーにつながる危険な敵。
4. **増殖兵（スポナー）**: 3ターンごとに、自身の左右の空きスペースに「歩兵」を生成する。
5. **地雷兵**: 倒された瞬間、周囲のプレイヤーキャラのスピードを0にする（弾をその場で停止させる）爆発を起こす。

## 5. ゲームサイクルのイメージ
1. チーム編成（初期はバウンス型1体のみ）
2. マップから進行ルート（次のステージ）を選択
3. ステージ開始（WAVEごとに敵が上部から補充される）
4. 全WAVEの敵を全滅でステージクリア -> 報酬獲得 -> 2へ戻る
5. ボス討伐でエリアクリア
